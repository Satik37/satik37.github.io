# satik37.github.io

This is the repository for my personal website – a developer portfolio built with **Vite**, **React** and **Tailwind CSS**, deployed on **GitHub Pages**.

> ⚠️ **Work in progress – full redesign in progress**  
> The site is currently being rebuilt from the ground up: layout, content, and structure are all in active development.  
> Some sections may be incomplete, empty, or look like drafts while I iterate.

---

## Goals of this project

This site is meant to be:

- My technical portfolio as a **frontend / mobile developer**
- A sandbox to experiment with modern tooling
- A place to showcase projects, experiments, and case studies (including web, mobile, and game‑related work)

---

## Tech stack

- **Build tool:** [Vite](https://vite.dev/)
- **UI library:** [React](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Hosting:** [GitHub Pages](https://pages.github.com/)

---

## Running the project locally

**Requires**
- [Node.js](https://nodejs.org/) installed (LTS version recommended)
- [npm](https://www.npmjs.com/) (comes with Node)

---

## Deployment

The site is automatically deployed to GitHub Pages using **GitHub Actions**:

- Every push to the `main` branch:
  - installs dependencies
  - runs `npm run build`
  - uploads the `dist/` folder as the GitHub Pages artifact

Live site:  
**https://satik37.github.io**
